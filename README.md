# 📍 Sistema de Consulta de Endereços com ViaCep

Este projeto foi desenvolvido como exercício da faculdade de **Desenvolvimento de Software**.  
O objetivo é criar um sistema simples para consultar endereços através da **API ViaCep**, com integração ao banco de dados **PostgreSQL**.

---

## ⚙️ Funcionalidades
- 🔹 Buscar endereço pelo CEP (utilizando a API ViaCep)  
- 🔹 Inserir endereço no banco de dados  
- 🔹 Listar todos os endereços cadastrados  
- 🔹 Menu interativo com `switch case` e `while`  

---

## 🛠️ Tecnologias Utilizadas
- **Java**  
- **PostgreSQL**  
- **JDBC** (para conexão com o banco)  
- **API ViaCep**
